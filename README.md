# 🌦️ Weather App

This is a simple and responsive Weather App built using HTML, CSS, and JavaScript.

## 🚀 Features
- Get real-time weather information
- Search weather by city name
- Clean and simple user interface
- Fast and lightweight

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Weather API

## 📸 Preview
This app displays current weather details like temperature, location, and conditions.

## 💡 How It Works
Enter a city name and the app fetches live weather data using an API and displays it instantly.

## 📌 Future Improvements
- Add 5-day forecast
- Improve UI/UX design
- Add error handling for invalid city names
